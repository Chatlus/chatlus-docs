Installation
------------

To use Chatlus, install and run the .apk file provided.

.. _usage:

Usage
-----

Onboarding
----------

1. Log-in to your profile, or sign up if you do not have one.
2. You will be prompted to join a local chatroom if you have not already.
3. Select your desired chatroom on the map (currently limited to area range), and press **Join**.
4. Congratulations! You can now use Chatlus freely.
